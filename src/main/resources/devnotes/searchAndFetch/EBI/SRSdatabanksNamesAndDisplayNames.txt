Databanks and tools supported in SRS

This page lists the databanks and tools officially supported in SRS for which BioWisdom provides regular updates and maintenance. Please remember that due to its dynamic nature SRS allows the integration of many more data sources (see the List of Public SRS installation for a a list of the ones we know about).

If you have a request for a new database or tool to be added to the list or are interested on how to integrate your own data into SRS please email srs_support@biowisdom.com
